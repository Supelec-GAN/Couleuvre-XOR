La documentation générée par Doxygen se présente sous deux formes différentes :

 -- Un ensemble de fichiers (inluant plusieurs .html) devant être interprétés par un navigateur. Pour cela, télécharger tous les fichiers du dossier  _html puis ouvrir index.html dans un navigateur.
 
 -- Un ensemble de fichiers (incluant plusieurs .tex) devant être compilés par un moteur Tex puis visualisés au format PDF. Pour cela, téléchager tous les fichiers du dossier _tex puis compiler les fichiers .tex à l'aide de pdflatex par exemple. Le plus simple est sans doute d'ouvrir les fichiers avec un IDE pour latex (TexMaker ou LateXila par exemple).

Les informations présentes dans les deux types de formats sont strictement indentiques.
